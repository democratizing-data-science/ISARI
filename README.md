# AI-Assisted Multimodal Research Master Dashboard

An interactive, single-file HTML dashboard for teaching, documenting, and integrating AI-assisted qualitative and mixed-methods research workflows. The dashboard brings together methodological frameworks, local AI reasoning, privacy-conscious analytic practices, and visual explanations for a suite of qualitative data science tools.

## Overview

This dashboard is designed as a public-facing educational and methodological resource for researchers, instructors, students, and research teams working with qualitative evidence, mixed-methods designs, and AI-assisted analytic workflows.

The dashboard emphasizes **Truly Equal Status Design (TESD)**: computational, visual, quantitative, and qualitative evidence should support one another without allowing automated outputs to replace researcher judgment, participant context, or interpretive accountability.

## What the Dashboard Covers

The dashboard includes tabs for:

- **Overview** — High-level orientation to the dashboard and its purpose.
- **Book** — Connection to the broader book project on data science and interactive visualization tools for qualitative evidence.
- **Truly Equal Status Design** — Epistemological framing for integrating qualitative, computational, and visual evidence.
- **Ethics and IRB** — Guidance on confidentiality, consent, anonymity, participant protection, and responsible AI-supported interpretation.
- **ISARI** — Local, offline AI-assisted scholarly reasoning for memoing, synthesis, visual interpretation, and evidence-grounded writing.
- **LACOID** — Latent code identification for long-form qualitative data.
- **MDCOR** — Machine-driven classification of short open-ended responses.
- **SENA** — Sentiment and Emotion Network Analysis for comparing emotional and sentiment structures across individuals, groups, and topics.
- **GeoStoryTelling** — No-code geo-contextualized storytelling for mapping qualitative narratives, multimedia, routes, and spatial context.
- **Integration** — Cross-framework synthesis through ISARI and privacy-preserving local AI workflows.
- **Prompt Lab** — Copy-ready prompts for analytic memoing, comparison, synthesis, visual interpretation, and writing support.
- **Technical Notes** — Practical implementation and workflow notes.
- **References** — Supporting literature, methodological sources, and software references.

## Key Features

- **Single-file HTML dashboard**  
  Runs in a browser without requiring a server.

- **Teaching-ready structure**  
  Organized into clear tabs for classroom use, workshops, research training, or project documentation.

- **Copy-ready prompts**  
  Includes structured prompts for using AI as a brainstorming, memoing, comparison, and writing-support partner.

- **Privacy-conscious orientation**  
  Emphasizes local processing, careful handling of sensitive qualitative evidence, and avoidance of unnecessary external uploads.

- **Visual methodological explanations**  
  Includes recreated diagrams, workflows, tables, and conceptual figures for offline reliability.

- **Cross-framework integration**  
  Shows how outputs from LACOID, MDCOR, SENA, GeoStoryTelling, and other qualitative data science tools can be integrated through ISARI while preserving researcher authority.

## How to Use

1. Download or clone this repository.
2. Open the dashboard HTML file in a modern web browser.
3. Use the sidebar navigation to move across methodological tabs.
4. Copy prompts from the Prompt Lab or framework-specific tabs.
5. Adapt the prompts, workflows, and examples to your own research context.
6. When working with sensitive data, follow your IRB, consent, confidentiality, and data-management requirements.

## Ethical Use

This dashboard treats AI as a **support tool**, not as a replacement for researcher interpretation.

Researchers should:

- return to original qualitative evidence before writing claims,
- distinguish descriptive patterns from interpretive conclusions,
- avoid uploading sensitive data to external systems unless approved,
- protect participant confidentiality and location privacy,
- document analytic decisions,
- use AI-generated text as provisional brainstorming,
- verify all outputs against source evidence.
